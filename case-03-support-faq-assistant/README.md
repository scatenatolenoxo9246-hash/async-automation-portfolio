# Case Study 03: Support FAQ Assistant Prototype

## Client scenario

A small business receives repeated customer questions by email, Instagram, WhatsApp, contact forms, or website chat. The team answers the same questions many times:

- Shipping and delivery
- Returns
- Product sizing
- Appointment booking
- Pricing and service details

## What this prototype does

This is a no-backend FAQ assistant demo. It searches a small knowledge base and returns the most relevant answer with suggested next actions.

It demonstrates:

- Simple knowledge-base search
- FAQ answer ranking
- Suggested follow-up prompts
- A clean customer-facing support widget layout

## Files

- `index.html`: demo page
- `styles.css`: interface styles
- `app.js`: search and answer logic
- `data/faq.json`: sample knowledge base

## Run

Open `index.html` in a browser.

No API key, database, or server is required for this prototype.

## Client value

This is useful for ecommerce stores, clinics, local service businesses, agencies, and founders who want a low-risk prototype before investing in a full AI chatbot or helpdesk integration.
