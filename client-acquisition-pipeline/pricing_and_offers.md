# Pricing And Offers

The first goal is to win a paid test task, not a large project.

## Recommended first offers

### Offer 1: Data cleanup test task

Best for:

- CSV/Excel cleanup
- Shopify product feed preparation
- CRM export cleanup
- Lead list formatting

Scope:

- Up to 300 rows or a clearly limited sample
- Cleaned output file
- Manual review notes
- Short handoff summary

Price:

- USD 80-200

### Offer 2: Shopify product workflow

Best for:

- Supplier catalog cleanup
- Shopify import-ready CSV
- Product tags/categories/SEO fields
- Duplicate SKU and missing field review

Scope:

- One product category or limited sample
- Shopify-ready CSV
- Before/after report
- Optional script if recurring

Price:

- USD 150-500 for test task
- USD 500-1500+ for larger catalog work

### Offer 3: Automation script/API helper

Best for:

- Webhook transformation
- API data pull
- Google Sheets workflow
- Recurring report script

Scope:

- One trigger/input
- One output
- Runnable script
- README and handoff note

Price:

- USD 150-500 for small helper
- USD 500-2000 for multi-step workflow

### Offer 4: FAQ assistant prototype

Best for:

- Ecommerce support
- Local service questions
- Internal knowledge-base search
- Pre-chatbot proof of concept

Scope:

- Small knowledge base
- Static demo or simple internal page
- Suggested escalation notes
- Handoff summary

Price:

- USD 150-500 for prototype
- USD 500-1500+ for integration

## Pricing rules

- Do not start with a big hourly commitment.
- Use fixed scope for the first task.
- Ask for inputs before quoting.
- Keep the first task small enough to finish in 1-3 days.
- Charge enough that the client treats it seriously.
- For the first overseas review/case, it is acceptable to price conservatively, but not free.

## Minimum viable paid test

Use this structure:

```text
I suggest starting with a small fixed-scope test:

Input: [file/workflow they provide]
Output: [one clear deliverable]
Timeline: [1-3 days after receiving inputs]
Price: [USD amount]
Included: one revision based on written feedback
Not included: production system integration, large data volume, ongoing maintenance
```

## When to quote hourly

Quote hourly only after:

- The client has paid once.
- Trust exists.
- Scope is exploratory.
- You can cap hours.

Suggested early hourly range:

- USD 15-25/hour for early trust-building work
- USD 25-45/hour after proof and stronger portfolio

Prefer fixed test tasks at the beginning.
