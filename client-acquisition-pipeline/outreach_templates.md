# Outreach Templates

Use these as source templates. Do not send them unchanged. Each message should include one specific public signal from the lead's website or profile.

## Core positioning

```text
I am a China-based developer helping agencies and small teams with async implementation work: CSV cleanup, Shopify product data workflows, CRM follow-up automation, API scripts, and lightweight AI-assisted support prototypes.

I work best with clear written requirements and can start with a small paid test task before anything larger.
```

```text

If a quick call is necessary, I can do it, but it would be helpful if you send the task context first: current input, desired output, tools involved, and any deadline.

```

## Message quality checklist

Before sending:

- Does it mention one specific public signal from the lead?
- Is it under 120 words when possible?
- Does it ask for a low-friction next step?
- Does it avoid fake urgency and overpromising?
- Does it clearly say small paid test task, not free work?
- Does it match what the user can actually deliver?
