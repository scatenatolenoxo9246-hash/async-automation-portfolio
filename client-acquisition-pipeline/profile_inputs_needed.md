# Profile Inputs Needed From User

Before we start real outreach, collect these inputs. Place the answers in `user_profile.md` later.

## Required

1. Display name in English
   - Example: `Alex Wang`
2. Contact email
   - Gmail is fine at the beginning.
3. Location wording
   - Recommended: `China-based developer`
4. Development stack
   - Python, Node.js, JavaScript, frontend, scraping, browser automation, APIs, Excel/CSV, Google Sheets, Shopify, etc.
5. Weekly availability
   - Example: `10-15 hours/week`, `20 hours/week`, or `full-time available`.
6. Communication boundary
   - Recommended: `written async communication preferred`.

8. Portfolio URL

10. Payment readiness

12. Preferred service direction
